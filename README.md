# 🤖 MockMate AI

An AI-powered mock interview platform built using **Flask** and **Google Gemini AI**. MockMate AI helps students and job seekers practice technical interviews by generating intelligent interview questions, evaluating responses, and providing personalized feedback.

---

## 🚀 Live Demo

https://mockmate-ai-1-amnv.onrender.com
---

## 📌 Features

- AI-generated technical interview questions
- Multiple interview roles
- Company-specific interview support
- Difficulty selection
- Interview progress tracker
- Personalized AI feedback
- Responsive user interface
- Secure API key management
- Cloud deployment using Render

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/MokshThakur/MockMate-AI.git
```

Go to the project directory

```bash
cd MockMate-AI
```

Install dependencies

```bash
pip install -r requirements.txt
```

Create a `.env` file and add your Gemini API key.

Run the application

```bash
python app.py
```

Open

```
http://127.0.0.1:5000
```
